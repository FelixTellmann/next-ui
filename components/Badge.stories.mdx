# Badge

Let's define a story for our `Badge` component:

We can drop it in a `Canvas` to get a code snippet:

<Canvas withToolbar>
  <Story name="positive" >
    {(args) => <Badge status="positive" onClick={linkTo('Header','Header')}{...args}>positive</Badge>}
  </Story>
</Canvas>

<ArgsTable story="positive" />

We can even preview multiple stories in a block. This
gets rendered as a group, but defines individual stories
with unique URLs and isolated snapshot tests.

<Canvas>
  <Story name="negative">
    <Badge status="negative">negative</Badge>
  </Story>
  <Story name="negative">
    <Button status="negative">Button</Button>
  </Story>
  <Story name="warning">
    <Badge status="warning">Warning</Badge>
  </Story>
  <Story name="neutral">
    <Badge status="neutral">neutral</Badge>
  </Story>

</Canvas>
